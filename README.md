## ✨ Features

- Select any Python function in VS Code, right click → **"generate help"**
- AI analyses your code and returns inline comments covering best practices, docs references, and improvement suggestions
- Comments are inserted directly above your function in the editor
- Rate limited to 10 requests per minute
- Backend deployed on Google Cloud Run
- CI pipeline with mocked tests

---

## ⚙️ How It Works

1. User selects a Python function in VS Code
2. Right click → **"generate help"**
3. Extension sends the code snippet to the FastAPI backend on Cloud Run
4. Backend builds a prompt and calls Google Gemini
5. Gemini returns structured JSON with `docs`, `summary`, and `improvements`
6. Extension parses the response and inserts formatted comments above the function

```python
# 📚 Docs: Python Requests Documentation - https://requests.readthedocs.io
# 📝 Summary: This function fetches user data from a REST API
# ✅ Improvement: Add a timeout parameter to requests.get()
# ✅ Improvement: Wrap in try/except to handle network errors
def fetch_user(url):
    response = requests.get(url)
    return response.json()
```

## 🛠 Tech Stack

- **Python** – FastAPI backend
- **TypeScript** – VS Code extension
- **Google Gemini API** – AI code analysis (`gemini-flash-latest`)
- **Google Cloud Run** – Serverless backend deployment (australia-southeast1)
- **Pydantic** – Request and response validation
- **SlowAPI** – Rate limiting (10 requests/minute)
- **pytest** – Backend testing with mocked Gemini responses
- **GitHub Actions** – CI pipeline on pull requests to main
- **vsce** – VS Code extension publishing

## 💻 Local Development

### Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

Visit `http://127.0.0.1:8000/docs` to test the `/educate` endpoint interactively.

Add a `.env` file in `backend/`:

```
GEMINI_API_KEY=your_key_here
```

### Extension

```bash
cd extension/educate
npm install
npm run compile
```

Press `F5` in VS Code with `extension/educate` open as the root folder to launch the Extension Development Host.

### Running Tests

```bash
cd backend
pytest test_main.py -v
```
